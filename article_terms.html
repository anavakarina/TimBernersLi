Список терминов

Обновлено: 8 ноября 2025
HTML (HyperText Markup Language)
Язык гипертекстовой разметки, используемый для создания веб-страниц. Разработан Тимом Бернерсом-Ли в 1990 году.
HTTP (HyperText Transfer Protocol)
Протокол передачи гипертекста, основа взаимодействия в Вебе. Определяет как сообщения форматируются и передаются.
URL (Uniform Resource Locator)
Унифицированный указатель ресурса - адрес веб-страницы или файла в интернете.
Всемирная паутина (World Wide Web)
Распределенная система взаимосвязанных гипертекстовых документов, доступных через интернет.
Веб-браузер
Программное обеспечение для просмотра веб-страниц, навигации по веб-сайтам и работы с веб-приложениями.
Веб-сервер
Компьютерная программа, которая обслуживает запросы по протоколу HTTP и предоставляет веб-контент.
Гипертекст
Текст, содержащий ссылки на другие тексты, что позволяет нелинейно организовывать информацию.
W3C (World Wide Web Consortium)
Организация, разрабатывающая и внедряющая технологические стандарты для Всемирной паутины.
Semantic Web
Концепция развития Веба, при которой информация снабжается метаданными, понятными компьютерам.
Open Data
Концепция, согласно которой определенные данные должны быть свободно доступны для использования и распространения.
