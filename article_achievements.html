Всемирная паутина (World Wide Web)

В марте 1989 года Тим Бернерс-Ли представил проект, который изменил мир. Изначально он назывался Mesh, но позже был переименован в World Wide Web.

Ключевые принципы Всемирной паутины:

  • Децентрализация - нет центрального контролирующего органа
  • Открытость - стандарты доступны всем
  • Бесплатность - не требует оплаты за использование

Язык гипертекстовой разметки HTML

HTML (HyperText Markup Language) был создан как простой язык для создания связанных документов. Первая версия содержала всего 18 элементов.

Протокол передачи гипертекста HTTP

HTTP (HyperText Transfer Protocol) - протокол прикладного уровня для передачи гипертекстовых документов. Первая версия HTTP/0.9 была крайне простой:

  • HTTP/0.9 (1991) - Только метод GET, нет заголовков, только HTML
  • HTTP/1.0 (1996) - Добавлены методы POST, HEAD, заголовки, коды статуса
  • HTTP/1.1 (1997) - Постоянные соединения, кэширование, виртуальные хосты

Первый веб-браузер и сервер

Бернерс-Ли создал не только стандарты, но и первые инструменты для работы с Вебом:

WorldWideWeb Browser

Первый браузер был также редактором - можно было создавать и редактировать страницы прямо в браузере. Он работал на компьютере NeXT.

Первый веб-сервер

Сервер httpd (HTTP Daemon) был написан на Objective-C и запускался на том же компьютере NeXT.

Стандарты и W3C

В октябре 1994 года Бернерс-Ли основал Консорциум Всемирной паутины (W3C) в Массачусетском технологическом институте.

Основные цели W3C:

  1. Универсальный доступ - сделать Веб доступным для всех людей
  2. Семантическая паутина - сделать информацию понятной компьютерам
  3. Доверие - создать безопасную и надежную среду

"The power of the Web is in its universality. Access by everyone regardless of disability is an essential aspect."

— Tim Berners-Lee

Важные вехи в развитии стандартов:

  • 1995 - HTML 2.0
  • 1997 - HTML 4.0
  • 1998 - CSS 2
  • 2014 - HTML5 становится рекомендацией
